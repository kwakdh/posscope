## 운영 반영 완료 (배포된 기능)

### 상품 및 카테고리 예외처리
- **재고 품절** → 알림 팝업 ("재고 수량이 부족합니다. 재시도")
- **판매시간 종료 상품** → 알림 팝업 ("판매시간이 아닙니다. 재시도")
- **삭제된 상품/카테고리/옵션** → 알림 팝업 ("정보를 찾을 수 없습니다. 재시도")
- **우선순위**: 삭제 > 품절 > 판매시간 종료 순으로 팝업 1개만 노출
- 레이어팝업 내 터치 없을 경우 → 관리자 기본설정의 주문 초기화 시간 따라감
- **갱신 시점**: 홈으로 되돌아왔을 때 갱신 (기존 정책 유지)

### 상품 옵션 처리
- 체크박스 선택된 채로 상품담기 화면 노출
- 옵션 필수 여부 "선택": 선택 해제 가능
- 옵션 필수 여부 "필수": 해제 가능하나 담기 버튼 비활성 (옵션 개수 = 선택 개수 전제)

### [D2307] 관리자 설정 내 로그전송 버튼
- 취소/X: 팝업 닫음, 로그전송 취소
- 확인: 팝업 닫음, 로그전송 완료

### [D2508] 키오스크 업데이트 버튼
- 일반 업데이트: 취소 버튼 있음, 업데이트 선택 가능
- 강제 업데이트: 취소 버튼 없음, 무조건 업데이트

### [D-2600] 일회용품 사용 모달 설정
- 경로: 관리자 설정 > 기본설정 > 일회용품 요청
- 디폴트: 물어봄 (포장 주문 시 일회용품 여부 모달 노출)
- 물어보지 않음: 매장이용과 동일하게 주문현황 내 일회용품 영역 미노출 (disposable=null)

### [D-2149] 키오스크 리더기 재연결 모달
- 단말기 연결 실패 시: "결제 장비 연결에 실패했습니다. 카운터에 문의해주세요."
- [장비 설정 이동] 버튼 제공

### [D-2700] 전자+종이영수증 출력 옵션
- AS-IS: 라디오버튼 (단일 선택) → TO-BE: 체크박스 (복수 선택 가능)
- 디폴트: 전체 체크 해제 (둘다 미표시)
- 출력 불가 리더기 연결 시 종이영수증 체크박스 비활성화
- 종이+전자 모두 체크 시: 결제완료 모달 내 선택 버튼 추가

### 키오스크 출력설정
- 경로: 키오스크 관리자 > 출력설정
- 결제기기/프린터/일체형 포스별 종이영수증·주문서 체크박스 설정
- 체크박스 하나라도 on → 해당 영역 노출
- 바로가기: 키오스크 관리자 > 기본 설정 > 결제화면 > 영수증 발급으로 라우팅
