# 베리포스 (BerryPOS) — 명칭 주의

**명칭**: 항상 **베리포스**로 표기. "베리폰"은 틀린 명칭이므로 사용 금지.

**Why:** 포스앱 기능 추가·수정 시 베리포스도 함께 기획 필요 — 기획 요청 시 "베리포스도 함께 기획할까요?" 먼저 물어볼 것.

- KCP POS 앱의 **세로형(portrait) 버전**

- KCP POS 기능의 **60~70%** 포함 (일부 기능 미지원)
- 기능 추가·수정 시 **포스앱 + 베리포스 동시 기획** 필요
