## 상단 탭 구성 (4개)

상품 / 테이블 / 주문현황 / 마이페이지

우측 상단 버튼: **키오스크** (모드 전환) / **영업중** / 고객센터 안내 아이콘 (헤드셋 모양)

---

## 1. 상품 탭

- **상품 찾기**
- **기능**
  - 상품 정보 수정 / 상품 순서 변경
  - 카테고리 수정 / 카테고리 순서 변경 / 카테고리 선택 / 카테고리 추가
  - 타이틀 추가 / 타이틀 수정 / 타이틀 선택
  - **상품 선택** ← 핵심
  - 상품 추가/수정/복사
- **임시저장**
- **-/+ 버튼**
- **상품 별 할인** : 금액별 / 퍼센트별 / 자주쓰는 할인 비율
- **추가금액 입력**

---

## 2. 테이블 탭

- **홈**
- **테이블 정보**
  - 테이블 인원 추가 / 주문하기 / 자리이동 / 테이블 합치기 / 테이블 체류시간
- **기능**
  - 구역 정보 수정 (구역명/테이블 수) / 구역 순서 변경
  - 테이블명 수정 / 테이블 순서 변경
  - 테이블 구역 변경은 불가(?)
- **테이블 선택**
  - 테이블 주문 현황
  - 상품 선택 삭제 / 상품 선택 추가 / 동일 상품 접수
  - 중간계산서 인쇄
- **추가금액 입력**

> ❌ QR 코드 발급, 태블릿오더 연동 기능 제거됨

---

## 3. 주문 현황 탭

- **장비 설정 퀵버튼**
- **접수 내역 (홈)** → 접수 목록 선택
  - 주문 번호 / 테이블 명 / 주문 상세 / 주방프린트 인쇄 / 상태 변경(취소/완료)
- **완료 내역 (홈)** → 완료 목록 선택
  - 주문 번호 / 테이블 명 / 주문 상세 / 주방프린트 인쇄 / 상태 변경(접수)
- **취소 내역 (홈)** → 취소 목록 선택
  - 주문 번호 / 테이블 명 / 주문 상세

---

## 4. 마이페이지 탭 (1+2 통합, 현행 기준)

좌측 패널 상단: 가맹점명 / 실시간 매출 / **영업 시작/종료** 버튼

**메뉴 리스트 (순서대로):**

1. **결제 내역**

2. **우리 매장 분석** (웹뷰)
   - 매출 달력 / 매출 내역 분석 / 시간 별 분석 / 상품 별 분석 / 회원 & 포인트

3. **회원 관리**
   - 회원 리스트 / 회원 등록 / 포인트 설정

4. **네이버 플레이스** N 신규 (연동중 뱃지)

5. **모드별 설정 (QR 배달)** N 신규
   - **QR 오더** N 가맹점 전용
     - 테이블주문 (QR/NFC) / 픽업주문 (QR) / QR 코드 관리 / 운영 시간 설정
   - **배달 설정** 비가맹점 뱃지
   - **비밀번호 설정**

6. **상품 관리 (재고, 옵션)**
   - 상품 옵션 관리 / 재고 관리

7. **사용자 관리**

8. **장비 설정**
   - **결제 기기 설정** (펼침)
     - 카드 단말기 설정 (케이블/네트워크 탭, 단말기 포트·속도·버전·VAN·TID, 단말기 연결 버튼)
     - 카드 리더기 설정
   - 프린트 설정
   - 바코드 스캐너 설정
   - 출력 설정
   - 로그 전송

9. **배달 설정**

10. **기본 설정**
    - 버전 정보 (앱 정보 + 기기 정보: 기종/운영체제/OS버전/오픈소스 라이선스)
    - 할인 설정
    - 효과음 설정
    - 서비스 수신 설정
    - 비상모드
    - 매장 변경
    - 로그 아웃
    - 하단 링크: 개인 정보 처리 방침 / 서비스 약관

하단 버튼:
- **가맹점 신청** — 비가맹점일 때만 노출 / 가맹점이면 미노출
- **이용 가이드** — 항상 노출

> ❌ 키오스크 설정(모드별 설정) 메뉴 제거됨
> ❌ 공지사항, 매장 설정 별도 메뉴 없어짐 (공지사항은 우측 상단 배너로 이동)

---

## 6. 결제 탭

- **결제 화면 구성**
  - 분할 결제 / 할인 설정 / 포인트 적립
  - 카드 결제 / 현금 결제 / 간편 결제 / **배달 결제 (준비)** / 결제 관련 신청
- **홈**
  - 매장/포장 여부
  - 상품 선택 → 상품 리스트 / 상품 상세 옵션 / 장바구니
  - 결제 → 주문 알림 발송 / 주문 내역 확인 / 결제 방법 선택(카드/간편결제) / 포인트 적립 / 결제 안내 / 영수증 설정

---

## 5. 모드 변경 (키오스크 버튼)

우측 상단 **키오스크** 버튼으로 진입.
- **키오스크 모드 전환** 기능만 존재

> ❌ 관리자 설정 하위 메뉴(결제내역, 장비 연동 설정, 시작 화면 설정 등) 별도 없음
