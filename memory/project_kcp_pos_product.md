# KCP POS+ 클라우드 POS 소개서 내용

**제조사:** NHN KCP Corp. / 브랜드명: LYNK  
**성격:** 국내 VAN사 최초 모바일 클라우드 포스

**Why:** 기획서 작성, 기능 설계, 사용자 설명 시 제품 이해도를 높이기 위한 레퍼런스.

스마트폰, 태블릿, 노트북만 있으면 언제든지 설치 가능  
어떠한 디바이스와 운영체제에서도 구동 / 야외 무선 사용 가능  
윈도우 & APP 인증 모두 완료

## 주요 기능

### 상품 관리
- 상품명만 입력하면 10초 만에 메뉴 등록
- 가격 수정 5초
- 엑셀파일 상품 일괄 등록 지원
- 재고 수량 설정 / 재고 기반 품절 처리

### 테이블 관리
- 후불 매장 테이블 커스텀 (테이블명·좌석수·크기 설정)
- 화장실·입구 등 공간 요소도 편집 가능

### KDS (주방 디스플레이)
- 태블릿·스마트폰만으로 KDS 구축
- 좌우로 밀어 완료/취소 처리
- POS 테이블 및 키오스크 포장 정보 확인

### 회원 관리
- 회원 등록 / 회원 리스트
- 회원별 포인트 및 결제 히스토리

### 매출 분석 (우리매장 분석)
- 매출달력 / 매출 내역 분석 / 시간별 매출 / 상품별 매출 / 회원 & 포인트 / 매출 손익 분석

### QR오더
- 포스 앱 주문 실시간 동기화
- 포스에 등록된 메뉴 그대로 사용
- NHN KCP 결제망 (온라인 신용카드 TRAFFIC 점유율 1위)

### 배민 직연동
- 배달의민족 직연동 (윈도우 버전)
- 요기요 & 쿠팡이츠 추후 제공 예정

### 관리자 페이지 (백오피스)
- 가맹점·대리점·프랜차이즈용 WEB 관리자페이지
- URL: https://plus.kpos.store
- 매출/회원/사용자/상품(엑셀업로드)/재고관리

### 부가 서비스 (무료)
- 매출리포트 카카오 알림톡
- 바로할인 (결제 시 할인키 제공)
- 전자영수증
- 매장 10개까지 생성
