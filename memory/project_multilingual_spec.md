## 파트너 다국어 설정 메인화면 Descriptions (항목 1~c)

1. **'다국어 설정' 메뉴 추가** — 노출 사용자 타입: 전체
2. **'검색유형' 셀렉트박스** — 브랜드명 or 매장명 선택 시 해당 브랜드/매장의 다국어 설정 화면(연두색 박스) 노출
3. **가이드 문구** (검색유형 선택에 따라 분기)
   - 선택 전: `'매장 또는 브랜드를 선택해 주세요.'`
   - 브랜드 선택 시: `'"{브랜드명}" 매장 상품 목록입니다. 해당 브랜드의 다국어 설정을 관리 할 수 있습니다.'`
   - 브랜드 있는 매장 선택 시: `'"{브랜드명}" 브랜드의 "{매장명}" 매장 상품 목록입니다. 해당 매장의 다국어 설정을 관리 할 수 있습니다.'`
   - 브랜드 없는 매장 선택 시: `'"{매장명}" 매장 상품 목록입니다. 해당 매장의 다국어 설정을 관리 할 수 있습니다.'`
4. **탭** — 카테고리, 타이틀, 상품, 옵션 순 노출
5. **갯수 노출** — 현 페이지에 노출 중인 상품 개수
6. **상품 다국어 설정**
   - [6-1] 미설정: '-' 표시
   - [6-2] 설정됨: 문구 노출
   - [6-3] [A 언어설정] 키오스크 토글 비활성화 or 해당 언어 토글 비활성화 시: 비활성화 처리
7. **페이징 셀렉트박스** — 디폴트: 30 / 옵션: 30, 50, 100, 200
b. **저장/취소 Btn**
   - 저장: 저장 + 연두색 박스로 변경 + 토스트 (2s) `"저장되었습니다."`
   - 취소: 저장 안함 + 연두색 박스로 변경
   - 저장 없이 다른 탭 이동 or 취소 클릭 시: 얼럿 팝업 노출
c. **AI 자동완성 Btn (베타)**
   - 클릭 → 얼럿 팝업 → 확인 클릭 시, 주황색 박스 내 텍스트 자동 번역 노출 (수동 편집 가능)

## 파트너 언어설정 화면 [A] Descriptions (항목 1~3)

1. **모드 선택 토글**
   - 현재: '키오스크' 토글만 제공 (추후 QR오더 등 추가 예정)
   - 디폴트: 비활성화 / 활성화 시 [A-1] 화면으로 변경
2. **언어 관리 토글**
   - 제공 언어: 영어, 중국어, 일본어
   - 디폴트: 전체 활성화
   - 활성화된 언어만 기본언어 셀렉트박스[3]에 노출됨
3. **기본언어 셀렉트박스**
   - 한국어: 항상 존재 (디폴트)
   - 영어/중국어/일본어: [2] 토글 활성화 시에만 셀렉트박스에 노출
   - ex) 영어+일본어만 활성화 → 셀렉트박스 = 한국어, 영어, 일본어

## 이력 문구 노출 정책

전제: 언어 사용 토글 활성화 유무에 따라 문구 상이 (영어만 사용 시 영어 항목만 노출)

형식:
```
★카테고리 수정 문구> [카테고리 수정]
  영어 : {영어 입력 내용}
  중국어 : {중국어 입력 내용}
  일본어 : {일본어 입력 내용}

★타이틀 수정 문구> [타이틀 수정]
  영어 : {영어 입력 내용}  ...동일 형식

★상품 수정 문구> [상품 수정]
  영어 : {영어 입력 내용}  ...동일 형식

★옵션 수정 문구> [옵션 수정]
  영어 : { 옵션그룹명 영어 / 옵션명 영어, 옵션명 영어... }
  중국어 : { 옵션그룹명 중국어 / 옵션명 중국어... }
  일본어 : { ... }
```

## 키오스크(포스앱) 언어선택 Btn 정책

**노출 조건:**
- 파트너>다국어설정>언어설정>키오스크 토글이 활성화된 경우에만 노출
- 활성화된 언어 토글 수에 따라 언어 btn 수 결정
  - ex) 일본어만 활성화 → '한국어/일본어' btn만 노출
  - ex) 아무 언어 없음 → 다국어 btn 미노출

**홈화면으로 돌아갈 때 언어 기준 (3가지):**
1. 타임아웃 초기화 → 디폴트 언어 (파트너>다국어설정>언어설정>기본언어 값)
2. 결제 후 홈화면 → 디폴트 언어
3. 홈버튼 클릭 시 → [키오스크 관리자>기본설정>주문초기화시간] 동안은 이전 선택 언어 유지 → 30초 후 디폴트 언어

**UI:** 기획서 UI와 달라도 됨 (UI팀 확인 사항)

## Why
키오스크 다국어 기능은 파트너(어드민)에서 언어 설정을 관리하고, 키오스크(포스앱/베리포스)에서 그 설정을 그대로 반영하는 구조.
파트너에서 설정한 내용(활성 언어, 기본언어)이 키오스크 화면에 직접 연동됨. AI자동완성은 베타 기능.
